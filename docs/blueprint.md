# **App Name**: Sacred Echoes

## Core Features:

- Navigation Bar: Fixed-position navigation bar with logo, choir name, and search icon.
- Section Banners: Visually distinct banners featuring choir photos and information.
- Responsive Layout: Ensure the website is responsive and adapts to different screen sizes.
- Search Functionality: Enables users to search for content within the website.

## Style Guidelines:

- Primary color: Deep Black (#000000) for elegance and solemnity.
- Secondary colors: Pure White (#FFFFFF) for contrast and Mustard Gold (#C4A850) for highlights.
- Accent color: Subtle border (#333333) for division, link text (#D14936) for functional use, purple gradient for panel backgrounds.
- Headline font: 'System UI, San Francisco, Roboto, sans-serif' (sans-serif) for modern display headlines.
- Body font: 'Georgia, Times New Roman, serif' (serif) for descriptive paragraphs.
- Use line art icons in dark gray.
- Sections with generous margins and 4rem to 6rem padding for an airy design and lots of negative space (black or white).
